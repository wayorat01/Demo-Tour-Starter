## ✅ สิ่งที่ทำเสร็จ

- สำเร็จภารกิจ **TASK-004: ย้ายข้อมูล Seed ออกจาก Git** 
- ย้ายระบบ SeedDB ไปใช้งาน S3 (Wasabi) ทั้งหมด เพื่อแก้ปัญหา Git repository บวม (จากการ track `seed-data.tar.gz` สะสมมาเรื่อยๆ จนรวมกว่า 231MB ต่อก้อน)
- ลบการเก็บไฟล์และการดึงข้อมูลก้อน Binary ออกจากโฟลเดอร์ `public/media/` สู่ S3 Bucket โดยตรง
- วางระบบกรองข้อมูล payload versions (`_versions`) ออกจากไฟล์ seed เพื่อให้ก้อน S3 สะอาด ไม่บวม และให้ UI ไม่มี collection ขยะ
- อัปเดต UI หน้า SeedDB Dashboard ให้ **สงวนสิทธิ์มีเฉพาะ user role `admin` เท่านั้นที่มองเห็นและคลิกได้** (เพื่อปกป้องฐานข้อมูลจากการใช้งานผิดประเภท)
- แก้ข้อความและ Wording รอบๆ หน้าต่าง UI จากการโหลด Git ให้อิงตาม S3
- เก็บโฟลเดอร์ `public/seed/` ลงใน `.gitignore` พร้อมสั่ง Untrack เรียบร้อย (ตกลงตามนโยบายว่ายังไม่รัน BFG ลบประวัติ 2GB ทิ้งเพื่อเลี่ยงผลกระทบกับทีม รอทำตอน Team Cleanup Day ทีเดียว)
- อัปเดต Documentation (`changelog/2026-03.md`, `TASK-004_seed_data_storage.md`) สะท้อนข้อตกลงและงานทั้งหมดเรียบร้อย
